# Bell Number

## Definition

The Bell Number is the number of partitions of a set.

Set:

### Set with 2 elements
Set:
```
ab
```
Partitions:
```
ab
a b
```
Bell Number: 2

### Set with 3 elements
Set:
```
abc
```
Partitions:
```
abc
ab c
ac b
bc a
a b c
```
Bell Number: 5
